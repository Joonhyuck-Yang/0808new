# Docker Commands for MSA Project

## Backend (Gateway)
# Build gateway image
docker build -t msa-gateway ./gateway

# Run gateway container
docker run -p 8080:8080 msa-gateway

# Run with docker-compose (backend only)
docker-compose up backend

## Frontend (Local development recommended)
# Frontend should be run locally with pnpm dev
# Docker is mainly for backend services

## Service Discovery
# Check running containers
docker ps

# View logs
docker logs <container_name>

# Stop all containers
docker-compose down

# Remove all containers and images
docker system prune -a
